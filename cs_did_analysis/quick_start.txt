# 保证金调整对期货市场波动率影响的双重差分分析快速入门

## 运行步骤

1. 确保已安装所有必要的R包:
   ```R
   install.packages(c("dplyr", "tidyr", "ggplot2", "data.table", "lubridate", 
                     "did", "readr", "stringr", "forcats", "fixest", "scales", "zoo"))
   ```

2. 运行完整分析:
   ```bash
   ./run_analysis.R
   ```

3. 查看结果:
   - 分析摘要: `cat output/did_analysis_improved/analysis_summary.txt`
   - 表格结果: `ls -la output/did_analysis_improved/tables/`
   - 图表结果: `ls -la output/did_analysis_improved/plots/`

## 预期输出

1. 分析摘要文件: 包含数据概览、主要发现和结论
2. 表格结果:
   - 保证金调整摘要
   - 处理状态摘要
   - 动态处理效应数据
3. 图表结果:
   - 动态处理效应图: 展示不同时期的处理效应
   - 波动率分布图: 比较处理前后的波动率分布

## 注意事项

- 完整分析可能需要几分钟时间
- 分析成功后，结果将保存在 output/did_analysis_improved 目录下
- 详细说明请参考 README_did_analysis.md
